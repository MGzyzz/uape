# UAPE — Образовательная платформа

UAPE — это веб-платформа для обучения программированию. Агрегирует курсы, плейлисты и каналы с YouTube, подбирает контент под каждого пользователя на основе его интересов и уровня подготовки.

---

## Что нужно установить заранее

Перед началом убедитесь, что на компьютере установлено:

| Программа | Зачем | Где скачать |
|-----------|-------|-------------|
| **Python 3.11+** | Запуск сервера | https://www.python.org/downloads/ |
| **Node.js 18+** | Запуск интерфейса | https://nodejs.org/ |
| **Poetry** | Управление пакетами Python | https://python-poetry.org/docs/#installation |
| **Git** | Скачать проект | https://git-scm.com/ |

> **Проверка:** Откройте терминал и введите `python3 --version`, `node --version`, `poetry --version`. Если показывает версии — всё установлено.

## Установка проекта

### Шаг 1. Скачать проект

```bash
git clone <ссылка-на-репозиторий>
cd uape
```

---

### Шаг 2. Настройка бэкенда (сервер)

```bash
cd back_end
```

**Установить зависимости:**
```bash
poetry install
```

**Создать файл `.env`** в папке `back_end/` со следующим содержимым:
```
YOUTUBE_API_KEY=ваш_ключ_от_youtube_api
```

> Ключ YouTube API получается бесплатно в [Google Cloud Console](https://console.cloud.google.com/). Нужно создать проект и включить "YouTube Data API v3".

**Применить миграции (создать базу данных):**
```bash
poetry run python3 manage.py migrate
```

**Создать аккаунт администратора:**
```bash
poetry run python3 manage.py createsuperuser
```
Введите имя пользователя, email и пароль — они нужны для входа в панель управления.

**Заполнить базу данных начальными данными:**
```bash
poetry run python3 seed_sections.py
```

---

### Шаг 3. Настройка фронтенда (интерфейс)

```bash
cd ../front_end
```

**Установить зависимости:**
```bash
npm install
```

**Создать файл `.env`** в папке `front_end/` со следующим содержимым:
```
VITE_API_URL=/api
```

---

## Запуск

Нужно запустить **два** процесса одновременно — в двух отдельных окнах терминала.

**Терминал 1 — сервер (бэкенд):**
```bash
cd back_end
poetry run python3 manage.py runserver
```
Сервер запустится на http://localhost:8000

**Терминал 2 — интерфейс (фронтенд):**
```bash
cd front_end
npm run dev
```
Интерфейс откроется на http://localhost:5173

## Открыть в браузере

- **Сайт:** http://localhost:5173
- **Панель управления (админка):** http://localhost:8000/admin

## Панель управления

Через панель управления (http://localhost:8000/admin) можно:

- Добавлять **каналы** — вставить ссылку на YouTube-канал, данные подтянутся автоматически
- Добавлять **плейлисты** — вставить ссылку на YouTube-плейлист
- Добавлять **видео** — вставить ссылку на YouTube-видео
- Управлять **разделами** на главной странице
- Просматривать список **пользователей**

## Частые вопросы

**Сервер не запускается — ошибка про порт?**
Значит порт 8000 занят другой программой. Запустите на другом порту:
```bash
poetry run python3 manage.py runserver 8001
```
Тогда в файле `front_end/.env` измените:
```
VITE_API_TARGET=http://localhost:8001
```

**Изображения не загружаются?**
Убедитесь что бэкенд запущен — фронтенд берёт медиафайлы с него.

**Забыли пароль администратора?**
```bash
cd back_end
poetry run python3 manage.py changepassword <имя_пользователя>
```
